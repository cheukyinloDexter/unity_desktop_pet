FIRST add the following packages into your VCC project in "Manage Project" in this order:

1. Modular Avatar
https://modular-avatar.nadena.dev/

2. FaceEmo
https://suzuryg.github.io/face-emo/

SECOND in unity import the custom package "poi_Toon_7.3.50_UpTo_9.0.54.unitypackage"

THIRD, import the custom package "RabbitHoleMiku_v1.0.unitypackage"

Read the readme txt file!

Go to the prefabs folder and drag the prefab into the hierarchy/scene

Open the VRChat SDK menu and upload!

===

This model uses FaceEmo for its expressions system
It supports both gesture expressions and quick menu expressions

If you would like to change how the expressions are binded to the gestures,
Or create your own expressions using the available blendshapes,
Please refer to the FaceEmo website:

https://suzuryg.github.io/face-emo/
